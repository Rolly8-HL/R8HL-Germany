R8HL_photogallery_panel v.1.0.3
Für PHP Fusion Version 7.2.x
R8HL Germany (http://rolly8-hl.de)
--------------------------------------
Importierte Quelle: von (http://highslide.com/)
---------------------------------------
1. Beschreibung
---------------------------------------
 8 verschiedene Slideshow 
Admin legt einen Standard fest.
User können selber wählen.
---------------------------------------
1. Installation v.1.0.3
---------------------------------------

Der Inhalt file müssen in das ROOT Verzeichnis der PHP-Fusion hoch geladen werden.

Gehe dann unter: System Admin => Infusionen => R8HL Photogallery Panel installieren

Ein Navigationslinks wird automatisch erstellt.

Zum Schluss den Admin dieser Infusion öffnen und die Grundeinstellung Ändern nach Bedarf.
---------------------------------------
2. Update auf v.1.0.1 &  v.1.0.2
---------------------------------------
Der Inhalt file müssen in das ROOT Verzeichnis der PHP-Fusion hoch geladen werden.

Gehe dann unter: System Admin => Infusionen => R8HL Photogallery Panel updaten

Viel Spaß!
